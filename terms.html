Privacy Policy

This Privacy Policy explains how our application collects, uses, stores, and protects Google user data in compliance with the Google API Services User Data Policy.

1. Data Accessed

Our application accesses the following Google user data through Google OAuth:

  • YouTube subscription information (to verify whether a user is subscribed to a specific channel)
  • Basic Google account identifier (such as a unique user ID)

2. Data Usage

The accessed data is used solely for the following purposes:

  • To verify whether a user is subscribed to a specific YouTube channel
  • To provide application functionality based on subscription verification (e.g. access control or role assignment)

We do not use Google user data for advertising, analytics, or marketing purposes.

3. Data Sharing

We do not share Google user data with any third parties. All accessed data is used exclusively within the application for its intended functionality.

4. Data Storage & Protection

User data is handled securely and protected using appropriate technical measures. OAuth access tokens and related credentials are stored securely and access is restricted to authorized systems only.

5. Data Retention & Deletion

Google user data is retained only for as long as necessary to provide the service. Users may request deletion of their data at any time by contacting us. Upon request, all associated user data will be permanently deleted.

6. User Rights

Users may revoke the application's access to their Google account at any time through their Google Account security settings.
